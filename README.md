# kasirsolo-pos

KASIRSOLO - Multi-App POS & Management Platform by PT Mesin Kasir Solo

## Package Manager

Project ini menggunakan **pnpm** sebagai package manager. Pastikan pnpm sudah terinstall di sistem Anda.

### Setup Awal

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

### Linting

```bash
pnpm lint
```

### Type Checking

```bash
pnpm typecheck
```

### Clean

```bash
pnpm clean
```
